== Description ==

This is a plugin to add custom widgets to [Elementor](https://github.com/pojome/elementor/) like :

* FS Citation
* FS Leaflet Map
* FS Playlist
* and more other widgets


== Installation ==

1. Installation must be done **manually**
2. Updates will be done automatically via the plugin updates of your site.
